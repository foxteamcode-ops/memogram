# Memogram

**Memogram** is a web-based platform for creating, publishing, and sharing memes and user-generated visual content.  
The project focuses on modern short-form and vertical media formats, as well as social interaction between users.

üåê **Live demo:** https://memogramm.vercel.app/

## Overview

Memogram is designed as a single-page web application that allows users to upload, browse, and interact with meme content through a clean and responsive interface.  
The platform is optimized for both desktop and mobile browsers.

---

## Features

- Uploading and publishing meme content  
- Vertical feed for content browsing  
- User profile pages  
- Search and content discovery  
- Interactive UI elements (viewing, sharing)  
- Modal-based post management  

---

## Technology Stack

Memogram is built using modern web technologies:

- **TypeScript**
- **React**
- Component-based architecture
- Service-based API interaction
- **Progressive Web App (PWA)** support
- Cloud deployment with automatic updates

---

## Availability

Memogram is available online and does not require installation.  
A modern web browser and internet connection are sufficient to use the platform.

---

## Purpose

The project is intended for users interested in:
- creating and sharing memes;
- consuming entertainment-focused visual content;
- experimenting with modern web-based social media platforms.
